## ✍️ Adding Blog Guides

All blog content is in `post.json`. Add a new object to the `posts` array:

```json
{
  "id": "7",
  "slug": "your-guide-slug",
  "title": "Your Guide Title",
  "date": "7 May 2026",
  "dateiso": "2026-05-07",
  "category": "Beginner Guide",
  "emoji": "🔑",
  "featured": false,
  "readtime": "5 min read",
  "author": "Sarah Mitchell",
  "authorTitle": "Digital Safety Writer, FreeStrongPassword.com",
  "excerpt": "One or two sentence summary for the guide card.",
  "tags": ["tag1", "tag2"],
  "content": "<p>Full article in HTML. Use h2, h3, p, ul, ol, strong etc.</p>"
}
```

**Categories:** Beginner Guide · Security Tips · Recommendations · Emergency Guide

Set `"featured": true` on exactly one post — it appears as the hero article on the guides page.

## 🔗 Affiliate Slots

Three slots in `index.html` — search for `AFFILIATE SLOT`:

```
AFFILIATE SLOT 1 — Password Manager (e.g. NordPass, Bitwarden Premium)
AFFILIATE SLOT 2 — VPN Service (e.g. NordVPN, Surfshark)
AFFILIATE SLOT 3 — Antivirus / Security (e.g. Bitdefender, NortonLifeLock)
```

One sidebar slot in `blog.html` — search `AFFILIATE SLOT — Sidebar`.

Replace `href="#affiliate-N"` with tracking URL and add `rel="nofollow sponsored"`.
